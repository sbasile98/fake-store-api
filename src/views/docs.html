Autenticazione Basic Auth

Tutte le chiamate API richiedono autenticazione con email e password. Usa uno qualsiasi tra i 5 account di test.

Header HTTP: Authorization: Basic base64(email:password)
Esempi di utilizzo:
  1. curl: curl -u email:password URL
  2. Postman / Insomnia: Imposta Authentication Type su "Basic Auth" con le credenziali
  3. JavaScript (fetch): Header Authorization: Basic btoa('email:password')
  4. Python (requests): requests.get(url, auth=('email', 'password'))

Prodotti

Basic Auth Scrittura: Admin
GET /api/products Lista prodotti con filtri e paginazione

Query Parameters

pageNumero pagina (default: 1)
limitProdotti per pagina (default: 10, max: 100)
searchCerca in titolo, descrizione e brand
categoryFiltra per slug o ID categoria
minPricePrezzo minimo
maxPricePrezzo massimo
sortprice_asc, price_desc, rating, newest
curl -X GET \ -u mario.rossi@example.com:password123 \ {{BASE_URL}}/api/products
curl -X GET \ -u mario.rossi@example.com:password123 \ "{{BASE_URL}}/api/products?search=cuffie&minPrice=50&maxPrice=200&sort=price_asc&limit=5"
GET /api/products/all Tutti i prodotti (senza paginazione)
curl -X GET \ -u mario.rossi@example.com:password123 \ {{BASE_URL}}/api/products/all

curl -X GET \ -u mario.rossi@example.com:password123 \ {{BASE_URL}}/api/products/1

Body (JSON)

titleNome del prodotto (obbligatorio)
descriptionDescrizione (obbligatorio)

Formato Risposte

Le liste (prodotti, categorie, ordini, utenti) restituiscono direttamente un array JSON — ogni elemento e' un record separato. La paginazione e' negli header HTTP.

Lista (array diretto)
[
  {
    "id": 1,
    "title": "Cuffie Bluetooth",
    "price": 149.99,
    ...
  },
  {
    "id": 2,
    "title": "Smart TV 4K",
    "price": 599.99,
    ...
  }
]

Header paginazione:
  X-Total-Count: 50
  X-Page: 1
  X-Per-Page: 10
  X-Total-Pages: 5
Errore
{
  "success": false,
  "error": "Descrizione dell'errore"
}

Codici HTTP:
  400 - Richiesta non valida
  401 - Non autenticato
  403 - Accesso negato (non admin)
  404 - Non trovato
  500 - Errore del server
